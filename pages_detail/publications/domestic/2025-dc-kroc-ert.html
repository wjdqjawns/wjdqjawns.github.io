Introduction

This work presents a compact and ROS-compatible Electrical Resistance Tomography (ERT) measurement board for robotic skin applications. ERT-based robotic skin is a promising approach for large-area tactile sensing in physical human-robot interaction, but existing systems often depend on bulky measurement hardware and limited software interoperability. To improve practical integration into robotic platforms, this project focuses on both hardware miniaturization and direct compatibility with ROS2-based systems.

Compact ERT Measurement Board

The developed board uses a 16-electrode, 2-pole measurement scheme implemented with four 8-channel multiplexers and a Teensy 4.0 MCU. The PCB size is \(100 \times 65~\mathrm{mm}^2\), which allows compact internal integration into robotic systems, and a potentiometer is used to adjust the impedance level depending on the sensing setup. For efficient acquisition, an optimized base measurement pattern with 62 measurements is used. The measured values are normalized by injected current, expressed as \(Z = \frac{V}{I}\), and sequential filtering is applied to improve measurement stability during real-time operation.

ROS Compatibility

The system is designed for Ubuntu 22.04 and ROS2 Humble, with the Teensy 4.0 operating as an independent node through \(\mu\)ROS. Raw sensor voltages are converted into ROS-compatible message formats and transmitted through ROS topics, enabling real-time monitoring and modular processing. This architecture improves compatibility with broader robotic software systems and makes it easier to integrate tactile sensing into distributed ROS-based workflows.

Conclusion

This work improves the accessibility of ERT-based robotic skin by reducing hardware size and increasing software compatibility with ROS environments. The developed board demonstrates that compact embedded hardware can support real-time ERT sensing while remaining practical for integration into actual robotic platforms, contributing to tactile sensing research for physical human-robot interaction.
