# JBJ Portfolio Site Scaffold

## Structure
- `index.html`: home
- `pages/`: main pages
- `pages_detail/`: project and publication detail pages
- `assets/`: papers, posters, demos, slides, images, resume
- `data/`: project/publication metadata
- `components/`: sidebar and footer
- `scripts/`: include and render logic
- `styles/`: base, layout, component, page styles

## How to update
1. Add a new detail page in `pages_detail/`
2. Add related files in `assets/`
3. Add one entry in `data/projects.js` or `data/publications.js`
